## Respuestas de "Catálogo E-commerce: Landing Page y UX/UI con IA"
- **¿Cuál fue el prompt exacto que mejores resultados te dio?**  
Actúa como un Diseñador UI experto y Desarrollador Frontend senior. Estoy construyendo la Landing Page de mi cafetería llamada 'Café El Buen Grano' para mi práctica de e-commerce. Ya tengo mi estructura en HTML5 semántico que incluye un encabezado, una sección 'hero' de bienvenida y un catálogo con 3 tarjetas de productos (Espresso, Latte y Capuchino).Genera el código CSS moderno y optimizado siguiendo estas directrices estrictas:
Layout Responsivo: Usa CSS Grid para el catálogo de productos, de modo que se vea en una sola columna en celulares y cambie automáticamente a tres columnas en computadoras. Diseño de Tarjetas (Cards): Dale a las tarjetas bordes redondeados sutiles, mucho espacio en blanco (white space) y una sombra suave para transmitir confianza visual. Agrega un efecto :hover para que la tarjeta reaccione sutilmente cuando el usuario pase el mouse (ej. que se eleve un poco o cambie la sombra).Jerarquía y Botones: Crea botones de 'Añadir al pedido' llamativos con un color de acento fuerte (tonos café oscuro). Asegura que el precio sea muy visible y tenga un contraste alto (estándar AAA) para facilitar la lectura. Micro-layout: Usa Flexbox dentro de cada tarjeta para centrar perfectamente la imagen, el nombre y el precio. Paleta de Colores: Usa tonos inspirados en el café (cremas y café oscuro) con un diseño limpio y minimalista.
Finalmente entrégame únicamente el código CSS, limpio, bien organizado y con comentarios que expliquen cada sección.

- **¿Qué principio de usabilidad (UX) consideraste para diseñar los botones de "Añadir al carrito"?**  
Se aplicó el principio de **Jerarquía Visual**. Esto significa que los botones de "Añadir al pedido" tienen un color de acento más fuerte y un tamaño que los hace destacar sobre el resto del texto. El objetivo es reducir la carga cognitiva del cliente, permitiéndole identificar instantáneamente la acción principal de compra. 

## Respuestas de "Landing Page de Cafetería - Diseño Responsivo Básico"
- **¿Por qué utilizamos Flexbox para cambiar la dirección de la imagen y el texto (row a column) en lugar de intentar hacerlo todo únicamente con Grid?**  
Utilizamos Flexbox para cambiar la dirección de la imagen y el texto porque es ideal para ordenar elementos en una sola dimensión: columna en celular y fila en pantallas grandes. Grid funciona mejor para la estructura general de la página, pero Flexbox permite controlar con más facilidad la alineación interna del producto. Así el diseño se adapta sin romper la lectura ni duplicar código innecesario.
